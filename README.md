## Getting Started

1. Clone the repository:

   git clone <URL_REPO>

2. Open the repository:

   cd...

3. Install the dependencies:

   npm i or yarn

4. Install the dependencies:

   npm run dev

   # or

   yarn dev

   # or

   pnpm dev

   # or

   bun dev

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
